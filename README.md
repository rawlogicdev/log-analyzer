# Simple LogAnalyzer 📋🔍

[![Version](https://img.shields.io/badge/version-1.0-blue.svg)](https://github.com/rawlogicdev/simple-loganalyzer)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A lightweight, terminal-based utility designed to parse, filter, and extract critical error logs from messy log files. This tool helps developers and system administrators quickly isolate system bugs without manual searching.

---

## Key Features

- **Log Parsing:** Reads and updates raw log files directly from the command line.
- **Bug Filtering:** Automatically detects error/bug patterns and isolates them.
- **Clean Exports:** Extracts filtered bugs into a dedicated, clean text file for easy debugging.
- **Interactive CLI Menu:** User-friendly terminal interface for fast navigation without complex flags.

---

## 🛠️ How It Works

`Simple LogAnalyzer` processes your raw application logs through a clean, interactive flow:

```text
[Raw Log File] ──> [Interactive CLI Menu] ──> [Filters Applied] ──> [Clean export.txt]
```

Lekkie narzędzie konsolowe stworzone do parsowania, filtrowania i wyodrębniania kluczowych błędów z nieuporządkowanych plików logów. Narzędzie pozwala programistom i administratorom systemów na błyskawiczne odizolowanie błędów bez konieczności ręcznego przeszukiwania plików.

---

## Główne Funkcje

* **Parsowanie Logów:** Odczytuje i aktualizuje surowe pliki logów bezpośrednio z poziomu konsoli.
* **Filtrowanie Błędów:** Automatycznie wykrywa wzorce błędów (bugów) i izoluje je od reszty wpisów.
* **Czysty Eksport:** Wyodrębnia przefiltrowane błędy do osobnego, przejrzystego pliku tekstowego, ułatwiając debugowanie.
* **Interaktywne Menu CLI:** Przyjazny dla użytkownika interfejs tekstowy umożliwiający szybką nawigację bez wpisywania skomplikowanych flag.
